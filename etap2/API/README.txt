Do Api wykorzystuje nodejs npm :experss i sqlite3

Instalacja:

npm init -y w celu inicjalizacji package.json
npm install experss
npm install sqlite3

UWAGA 
plik JS musi nazywać się index inaczej nie będzie działać poprawnie 


API testowane jest za pomocą aplikacji POSTMAN, która umożliwia prostą i szybką weryfikację API.
